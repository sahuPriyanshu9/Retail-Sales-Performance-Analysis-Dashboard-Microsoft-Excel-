🎯 Project Overview (STAR Method)
📊 SITUATION

Business Context:
A retail business operating across multiple regions and product categories faced challenges in monitoring and analyzing sales performance effectively.

Key Challenges:
📁 Scattered Data: Sales data was not centralized
⏰ Manual Reporting: Time-consuming Excel-based reporting
📉 Limited Insights: No clear visibility of trends and KPIs
❓ Decision Gaps: Difficulty in identifying top products & regions
🔄 Static Reports: No interactivity for deeper analysis

Data Scope:
Thousands of transaction records
Multiple dimensions: Category, Region, Product, Time
Metrics: Sales, Quantity, Profit

🎯 TASK
Objective:
Design and develop an interactive dashboard to:

Track overall sales performance
Identify top-performing products and regions
Analyze monthly and yearly trends
Enable self-service analytics using filters
Automate reporting for faster decision-making

⚙️ ACTION
🔹 Phase 1: Data Preparation
Cleaned dataset (removed null values & duplicates)
Standardized data types (Date, Number, Text)
Created derived columns:
Year, Month
Revenue calculation
Structured data for analysis

🔹 Phase 2: Data Analysis
Built multiple Pivot Tables:
Sales by Category
Sales by Region
Monthly Trends
Top Products
Applied sorting, filtering, and aggregation

🔹 Phase 3: KPI Development
Developed key business metrics:
💰 Total Sales
📦 Total Quantity Sold
📈 Average Sales
🌍 Region-wise Performance
🏆 Top Products

🔹 Phase 4: Data Visualization
Created interactive charts:

📈 Line Chart → Sales Trends
📊 Bar Chart → Category Performance
📍 Region-wise Analysis
🏆 Top Product Analysis
🎯 Design Principles:
Clean & professional layout
Consistent color scheme
Easy readability
Business-friendly labels

🔹 Phase 5: Interactivity
Added slicers/filters:
Region
Category
Date
Enabled dynamic updates across dashboard

📈 RESULT
✅ Business Impact:
Reduced manual reporting time
Enabled real-time analysis
Improved decision-making
Increased visibility of sales performance
